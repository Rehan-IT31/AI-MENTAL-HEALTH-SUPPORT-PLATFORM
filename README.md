MindMate – AI Mental HealthSupport Platform

MindMate is an AI-powered mental health support platform designed to provide therapeutic emotional assistance, and personalized well-being features.

AI-powered mental health chatbot  

User authentication and secure login

Firebase authentication and database integration


Responsive and user-friendly interface  

Real-time support
